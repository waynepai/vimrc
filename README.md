# vimrc
設定主要參考下面<br>
amix - http://amix.dk/vim/vimrc.html<br>

安裝<br>
將.vimrc放到~/下以即將solarized.vim放到~/.vim/colors/下<br>
進入vimrc中打PluginInstall即可<br>
